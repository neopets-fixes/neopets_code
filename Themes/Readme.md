<h1>Installing the Theme</h1>
<p>
- Download Stylebot Extension (https://stylebot.dev/)
<br>- Stylebot > Options > New style
<br>- Enter url: <i>www.neopets.com</i>
 <br>- Copy & paste CSS into the body - <b>do not add style tags!</b>
